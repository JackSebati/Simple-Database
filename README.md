 CRM Database Project

 Description
This project is a Customer Relationship Management (CRM) database designed to manage customers, products, and orders.

Technologies Used
- SQL (Microsoft SQL Server / Azure SQL)
- Relational Database Design

Features
- Customer management
- Product catalog
- Order tracking
- Sales analysis queries

 Concepts Demonstrated
- Database normalization
- Primary and foreign keys
- Joins and aggregations
- Real-world business data modeling

 How to Run
1. Run schema.sql to create the database
2. Run sample_data.sql to populate data
3. Run queries.sql to test functionality
